    (inner) drawOverlay(options) → {google.maps.OverlayView}

    Draw an overlay in the Map and add it to the overlays collection. This method also fires a overlay_added event.

    Source:
    • overlays.js, line 11
    Parameters:
    Name Type Description
    options object

    The options object should contain:

    Name Type Description
    autoShow boolean

    Show the overlay inmediatly after being created.
    latitude number

    Latitude of the coordinate where the overlay is placed. If latitude and longitude are set, positionis ignored.
    longitude number

    Longitude of the coordinate where the overlay is placed. If latitude and longitude are set, positionis ignored.
    position google.maps.LatLng

    A google.maps.LatLng coordinate where the overlay is placed. If set, latitude and longitude are ignored.
    content string

    HTML that will be drawn in the overlay.
    layer string

    Id of any of the layers defined in google.maps.MapPanes:

    • floatPane
    • floatShadow
    • mapPane
    • overlayImage
    • overlayLayer
    • overlayMouseTarget
    • overlayShadow
    verticalAlign string

    Vertical align of the overlay (where the center is the coordinate latitude, longitude):

    • top
    • middle
    • bottom
    horizontalAlign string

    Horizontal align of the overlay (where the center is the coordinate latitude, longitude):

    • left
    • center
    • right
    horizontalOffset number

    Horizontal offset in pixels of the overlay.
    verticalOffset number

    Vertical offset in pixels of the overlay.
    Returns:
    Type:
    google.maps.OverlayView
