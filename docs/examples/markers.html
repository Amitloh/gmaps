Markers

GMaps#addMarker accepts any property from the native google.maps.Marker class.

latitude and longitude are required. You can also attach additional information with into the details property.

It also supports events as properties.

Note: If you want to show an Info Window, you must add:

infoWindow: {
  content: '<p>HTML Content</p>'
}
const map = new GMaps({
  element: '#map',
  latitude: -12.043333,
  longitude: -77.028333
});

map.addMarker({
  latitude: -12.043333,
  longitude: -77.03,
  title: 'Lima',
  details: {
    someNumber: 42,
    someUser: 'hpneo'
  },
  click: function(event) {
    alert('You clicked in this marker:\n' + JSON.stringify(this.details));
  },
  mouseover: function(event) {
    console.log(event);
  }
});
map.addMarker({
  latitude: -12.042,
  longitude: -77.028333,
  title: 'Marker with InfoWindow',
  infoWindow: {
    content: '<p>HTML Content</p>'
  }
});
