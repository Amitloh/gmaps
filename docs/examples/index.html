Examples

Find general use cases here:

  • Basic: Learn how to display a simple map with GMaps.
  • Markers: Add markers to your map.
  • Marker Clusterer: Group your markers in the map.
  • Polylines: Draw lines using coordinates.
  • Polygons: Draw simple or complex polygons using coordinates.
  • Geofences: Detect if a marker is inside or outside a polygon.
  • Context menu: Add a context menu to your map.
  • Custom controls: Add custom buttons to your map and perform personalized actions.
  • Elevation locations: Get elevation for coordinates.
  • Geocoding: Convert a coordinate to a human-readable address, or viceversa (requires Google Maps API key).
  • Geolocation: Locate your users using native browser's API.
  • KML: Display complex information using KML files.
  • Layers: Add Google's traffic, transit or bicycle layers.
  • Map Events: Bind events to your maps.
  • Map Types: Create tile-based custom map types usng third-party services.
  • Overlay Map Types: Create DOM-based custom map types.
  • Overlays: Add DOM elements on top of your map.
  • Routes: Display routes from an origin to a destination.
  • Travel routes: Display a step-by-step route navigation.
  • Render directions: Render directions for a route.
  • Static Maps: Create images from maps.
  • Styled Maps: Change map's styles.
