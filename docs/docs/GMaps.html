    GMaps

    GMaps is a wrapper for Google Maps JavaScript API. Its goal is to simplify Google Maps usage by performing complex tasks with fewer lines of code.

    Constructor

    new GMaps(options)

    Creates a new GMaps instance, including a Google Maps map.

    Source:
    • core.js, line 129
    Parameters:
    Name Type Description
    options object

    options accepts all the MapOptions and events listed in the Google Maps API. Also accepts:

    Name Type Description
    latitude number

    Latitude of the map's center.
    longitude number

    Longitude of the map's center.
    center google.maps.LatLng

    A google.maps.LatLng coordinate indicate the center. If set, latitude and longitude are ignored.
    element string | HTMLElement

    container where the map will be rendered.
    markerClusterer function

    A function to create a marker cluster. You can use MarkerClusterer or MarkerClustererPlus.
