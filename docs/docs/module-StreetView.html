    (inner) createPanorama(options) → {google.maps.StreetViewPanorama}

    Display a Street View Panorama for a position.

    Source:
    • streetview.js, line 10
    See:

    • options also accepts any option defined in the official documentation and any event defined in the "Events" section.

    Parameters:
    Name Type Description
    options object

    The options object should contain:

    Name Type Description
    latitude number

    Latitude of the coordinate where the panorama is placed. If not set, it takes the Map's center.
    longitude number

    Longitude of the coordinate where the panorama is placed. If not set, it takes the Map's center.
    position google.maps.LatLng

    A google.maps.LatLng coordinate where the panorama is placed. If set, latitude and longitude are ignored.
    Returns:
    Type:
    google.maps.StreetViewPanorama
