    (inner) drawSteppedRoute(options)

    Draw and travel a route automatically step by step. It uses the last route found by GMaps#getRoutes.

    Source:
    • routes.js, line 262
    Parameters:
    Name Type Description
    options object

    Accepts the same options as GMaps#travelRoute.

    Name Type Description
    origin string | google.maps.LatLng | google.maps.Place | google.maps.LatLngLiteral

    Location of origin.
    destination string | google.maps.LatLng | google.maps.Place | google.maps.LatLngLiteral

    Location of destination.
    route object

    A google.maps.DirectionsRoute object. If set, origin and destination are ignored.
    strokeColor string

    Color of the polyline. Can be hexadecimal or CSS name.
    strokeOpacity float

    Opacity of the polyline from 0.0 to 1.0
    strokeWeight integer

    Polyline width in pixels.
    start function

    Fired before the first step. It receives a single argument, which is a DirectionsRoute object.
    step function

    Fired each step in the route. It receives 2 arguments, which is a DirectionsStep object and the total length of the steps.
    end function

    Fired after the last step. It receives a single argument, which is a DirectionsRoute object.

    Get elevation information for locations or routes.

    Source:
    • routes.js, line 60
    Parameters:
    Name Type Description
    locations array

    Bi-dimensional array of latitudes and longitudes.
    path boolean

    If is true, makes a request along a path. If is false, only get elevation information on discrete locations.
    callback function

    Native callback function defined in google.maps.ElevationService ('Methods' section) (getElevationAlongPath if path is true, getElevationForLocations if path is false).
